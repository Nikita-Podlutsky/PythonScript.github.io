Wake-Up Word Detection

Этот проект направлен на создание и внедрение системы распознавания слов с использованием методов машинного обучения и обработки звука. Цель состоит в том, чтобы создать модель, которая сможет распознать конкретное слово или фразу из аудиовхода в режиме реального времени.

Решение каптчи

Данная система предназначена для решения изображений капчи с использованием машинного обучения. Она включает в себя извлечение признаков с помощью ResNet50, обнаружение объектов с помощью YOLO и систему кэширования для улучшения производительности. Система предоставляет веб-сервис на основе FastAPI.
